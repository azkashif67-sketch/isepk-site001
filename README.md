# ISE·PK — Integrated Security Engineering Pakistan

A high-performance, fully interactive website concept for **ISE·PK**, a commercial security engineering firm in Pakistan offering fire detection, CCTV surveillance, and access control systems.

## Live Site
[View on GitHub Pages](https://azkashif67-sketch.github.io/isepk-site001/)

## Features
- Animated hero with parallax + SVG trace line draw-in
- Tabbed service explorer (Fire / CCTV / Access) with spec sheets
- Filterable portfolio with modal detail view (6 projects, filter by system & city)
- Instant PKR cost estimator (site type × size × systems)
- Auto-advancing testimonials carousel with touch swipe
- FAQ accordion
- Scroll-spy nav, progress bar, mobile hamburger menu
- Battery/load-shedding callout specific to Pakistani commercial clients
- WCAG-friendly: focus rings, ARIA roles, reduced-motion support

## Stack
Pure HTML / CSS / JS — zero dependencies, zero frameworks.
